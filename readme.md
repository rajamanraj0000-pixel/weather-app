# 🌤️ Weather App

A beautiful weather application with animated backgrounds that change based on real-time weather conditions.

---

## 🌟 Features

- 🔍 Search weather by city name
- 📍 Auto-detect your current location
- 🌅 Dynamic landscape backgrounds for each weather condition
- 🌧️ Animated rain, snow, stars, and lightning effects
- 🌡️ Current temperature, feels like, min/max
- 💨 Wind speed, humidity, pressure, visibility
- 🌅 Sunrise and sunset times
- ⏱️ Hourly forecast
- 📅 5-day weather forecast

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/weather-app.git
   ```

2. Open the project folder:
   ```bash
   cd weather-app
   ```

3. Open `index.html` with **VS Code Live Server**

   > Right-click on `index.html` → Open with Live Server

---

## 🔑 API Setup

This app uses the **OpenWeatherMap API**.

1. Go to [openweathermap.org](https://openweathermap.org/) and create a free account
2. Get your API key from the dashboard
3. Add your API key in your backend/config file
